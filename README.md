# AI Resume Analyzer

An AI-powered Resume Analyzer built using Flask and Python.

## Features

- Upload PDF Resume
- Skill Detection
- Resume Score Calculation
- AI Recommendations
- Resume History Storage
- Bootstrap UI

## Technologies Used

- Python
- Flask
- SQLite
- PDFPlumber
- Bootstrap

## Run Project

pip install -r requirements.txt

python app.py
